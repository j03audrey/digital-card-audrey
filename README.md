## Features

- Responsive single-page business card
- Profile photo and personal introduction
- Contact email for collaboration and speaking invitations
- Social links: Instagram, Facebook, Threads
- Click-the-background color mood switcher

## Deploy To GitHub Pages

1. Push this repository to GitHub.
2. Open repository Settings -> Pages.
3. Under Build and deployment, choose:
	- Source: Deploy from a branch
	- Branch: `main` (or your default branch), folder `/ (root)`
4. Save and wait for deployment.
5. Your site will load from `index.html`, which routes to the business card page.

## Local Preview

Open `index.html` in a browser, or open `數位名片/digital-card-audrey.html` directly.
